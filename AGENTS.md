# AU MedTraker — AGENTS.md

## Project

Single Flutter app (`au_med`), NOT a monorepo. Medication tracker with Drift SQLite + Riverpod + GoRouter + Material 3.

## Entrypoints

- `lib/main.dart` — app bootstrap: initializes `AppDatabase` singleton, `NotificationService`, wraps in `ProviderScope` with `databaseProvider` override
- `lib/app.dart` — `AuMedApp` (ConsumerWidget), `MaterialApp.router` with Russian locale, light/dark themes, GoRouter
- `lib/src/router/router.dart` — `ShellRoute` with 3 bottom tabs (Medications, Today, Statistics) + full-screen routes for add/edit/archive/settings/dosage-history/log-editing

## Architecture

- **Database**: `AppDatabase` (Drift) — singleton via `getInstance()`, schema version 3, stored at `getApplicationDocumentsDirectory()/med_tracker.db`. Inline migrations in `MigrationStrategy`. Only `*.g.dart` is `database.g.dart` (committed to VCS).
- **Tables**: `MedicationsTable`, `MedicationLogsTable`, `DosageChangesTable`
- **DAOs**: `medicationsDao`, `logsDao`, `statisticsDao` — wired through `AppDatabase` properties, exposed via Riverpod providers
- **Providers**: Hand-written (NOT generated by riverpod_generator). `databaseProvider` must be overridden in `ProviderScope` (throws `UnimplementedError` otherwise). Theme/settings persisted via `SharedPreferences`.
- **Notifications**: `NotificationService` singleton, schedules repeating daily reminders at specific times. Cancels previous reminders for a medication before re-scheduling.
- **All Riverpod providers use `ref.read`** to access DAOs, NOT `ref.watch` — DB queries are manually triggered.

## Testing

- Single test file `test/widget_test.dart` — basic smoke test
- Tests call `AuMedApp` directly (no DB setup) — will fail if app logic requires DB at build time
- Run: `flutter test`

## Code quirks

- `namespace` and `applicationId` in Android config still set to `com.example.au_med` (placeholder)
- `flutter_launcher_icons` commented out in `pubspec.yaml` (launcher icon config left over)
- Accent colors: 4 hardcoded options (gray, dark-gray, blue, pink) in `AccentColorNotifier.colors`
- Date/time stored as ISO strings in DB (text columns)
- Medication `times` stored as comma-separated strings
- No CI, no pre-commit hooks, no task runner
